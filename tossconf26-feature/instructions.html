பதிவு செய்முறை படிகள்

TossHack'26 ஹேக்கத்தானில் பங்கேற்க இந்தப் படிகளைப் பின்பற்றவும்.

  1. GitHub பக்கத்திற்குச் செல்லவும்: முதன்மைப் பக்கத்தில் உள்ள 'GITHUB REPOSITORY' பொத்தானை அழுத்தவும் அல்லது அங்கு பட்டியலிடப்பட்டுள்ள பிரச்சனைகளில் (Problems) ஏதேனும் ஒன்றைக் கிளிக் செய்து நேரடியாக அதன் GitHub பக்கத்திற்குச் செல்லவும்.
  2. பிரச்சனையைத் (Issue) தேர்ந்தெடுக்கவும்: அங்கு கொடுக்கப்பட்டுள்ள பட்டியலில் நீங்கள் தீர்க்க விரும்பும் பிரச்சனையைத் தேர்ந்தெடுக்கவும்.
  3. விருப்பத்தைப் பதிவு செய்யவும்: அந்த Issue-ல் உங்கள் பெயரையும், உங்கள் குழு உறுப்பினர்களின் பெயர்களையும் குறிப்பிட்டு உங்களின் பங்கேற்பை Comment செய்யவும்.
